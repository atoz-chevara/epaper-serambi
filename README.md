# epaper-serambi
Serambi Indonesia ePaper Downloader

## Cara penggunaan

### Windows

 Buka Git Shell dan jalankan
 
		$ bash epaper-serambi.sh
		
### Linux

 Buka terminal dan jalankan
 
		$ chmod +x epaper-serambi.sh
		$ ./epaper-serambi.sh
